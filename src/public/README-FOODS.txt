IMPORTANT: To enable the Food Database with 1,700+ foods

Please copy /imports/foods.json to /public/foods.json

Steps:
1. Open /imports/foods.json
2. Copy all content (Ctrl+A, Ctrl+C)
3. Create new file: /public/foods.json
4. Paste all content (Ctrl+V)
5. Save file (Ctrl+S)
6. Refresh the app

The Food Database page will then load all 1,700+ food items!
