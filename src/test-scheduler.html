⏰ Email Scheduler Tester

Test your email scheduler endpoint and diagnose issues

📌 Purpose: This tool manually triggers the email scheduler to check for due reminders and future messages. Use it to test if your scheduled emails are being sent correctly.

🧪 Step 1: Test the Scheduler Endpoint

1 Click the button below to trigger the scheduler

This will check for any overdue reminders or future messages and attempt to send them.

📊 Step 2: Understanding the Results

2 What to look for:
  • success: true = Scheduler ran successfully
  • remindersSent = Number of reminder emails sent
  • messagesSent = Number of future message emails sent
  • errors: [] = List of any errors encountered

🔧 Step 3: Troubleshooting

⚙️ Cron Job Setup

For automated scheduling, set up a cron job:
  1. Go to cron-job.org
  2. Create a new cron job with the URL above
  3. Set it to run every 5-15 minutes
  4. Use GET method
  5. No authentication needed

📋 Checklist

Before using the scheduler, make sure:
  • ✅ SMTP credentials are set in Supabase Edge Functions
  • ✅ You have created a future message or reminder
  • ✅ The scheduled time has passed
  • ✅ You're logged into Resilio with a valid email
